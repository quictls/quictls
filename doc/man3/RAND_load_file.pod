=head1 NAME

RAND_load_file, RAND_write_file, RAND_file_name - PRNG seed file

=head1 SYNOPSIS

 #include <openssl/rand.h>

 int RAND_load_file(const char *filename, long max_bytes);

 int RAND_write_file(const char *filename);

 const char *RAND_file_name(char *buf, size_t num);

=head1 DESCRIPTION

These functions should not be used as random number generation no
longer stores or uses its state in external files.

=head1 RETURN VALUES

RAND_load_file() returns -1.

RAND_write_file() returns the number of bytes written (using RAND_bytes())
or -1 on error.

RAND_file_name() copies the name of a device that might be able to
generate random numbers or NULL1 if the buffer is not big enough.
